🎓 NovoLearn - Educational Management Platform
NovoLearn is a web-based educational management system developed as part of the PIDEV project for the 3rd Year Engineering Program at Esprit School of Engineering (Academic Year 2025-2026).

The platform enables educational institutions to efficiently manage and track student learning progress through an intuitive and robust web application.

🚀 Features:
-Student learning curve tracking

-Educational content management

-Progress monitoring and reporting

-User-friendly interface for administrators and educators

🛠️ Tech Stack:
-Framework: Symfony

-Language: PHP

-Database ORM: Doctrine

-Local Server: XAMPP

📚 Academic Context:
This project was developed in the context of the PIDEV (Projet de Développement) module, which is part of the engineering curriculum at Esprit School of Engineering, Tunisia.

-Institution: Esprit School of Engineering

-Program: Engineering Cycle – 3rd Year

-Class: 3A62

-Academic Year: 2025–2026

📦 Installation:
-Clone the repository

-Install dependencies via Composer

-Configure your database in .env

-Run migrations with Doctrine

-Start your local server (XAMPP or Symfony CLI)
